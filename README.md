# Controle de Gastos #

Este projeto tem como objetivo ajudar pessoas a organizar, acompanhar e entender melhor seus gastos financeiros de forma simples e prática.

A aplicação permite que usuários registrem despesas, categorizem seus gastos e tenham uma visão clara de para onde o dinheiro está indo, facilitando o controle do orçamento pessoal e a tomada de decisões financeiras mais conscientes.

A ideia central é oferecer um controle financeiro acessível, pensado para o dia a dia, sem complexidade excessiva, mas com estrutura suficiente para evoluir no futuro (relatórios, metas, usuários com perfis diferentes, etc.).
